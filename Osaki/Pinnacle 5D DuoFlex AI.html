Phone screen Mirroring on the controller
Phone screen Mirroring on the controller
The Pinnacle 5D DuoFlex AI has phone screen mirroring function. Users should make sure their phone support this function. And before using the function, they need to connect the same network for both their phone and controller. Measuring Guide
  • Calculate the doorway width by subtracting 2 inches from the smallest inner width of the door frame. (1 / 2 inch from each side)
  • Please use a standard measuring tape to measure the doorway.
  • If the door cannot be fully opened, open it as much as possible and measure the width from the edge of the door hinge.
  • If there is a rounded curve on the wall, calculate the inner width based on the most protruding point of the wall.
