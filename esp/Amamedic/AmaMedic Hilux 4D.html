Masajeador de Hombros
Masajeador de hombros ajustable para todo tipo de cuerpo. Apriete fácilmente empujando hacia adentro. 3 tamaños ajustables en cada lado. Restablezca el ancho predeterminado empujando ambos lados.
Masaje de Amasado de Pantorrillas
El reposapiernas superior tiene airbags para masaje circular de pantorrillas. Componentes añadidos para estiramiento y masaje de pies en puente y talón. Experiencia única.
Amasado de Pies
Bajo los pies, rodillos de reflexología giratorios ofrecen un excelente masaje. Los rodillos recorren los lados y arcos para una experiencia de amasado dimensional.
Reposapiés Extensible Automático
El componente de pies se extiende y retrae mientras escanea las piernas del usuario. Aplique presión para indicar la posición deseada. El control remoto permite ajuste automático del reposapiernas.
Altavoz Bluetooth HD
Bluetooth, sonido tipo anfiteatro, reduce el ruido, altavoces de cabecera de alta calidad. Comience con volumen bajo, aumente gradualmente.
Cargador USB y Cargador Inalámbrico
AmaMedic Hilux 4D tiene cargador inalámbrico en el apoyabrazos para smartphones. Requiere capacidad de carga inalámbrica.
Controlador de Pantalla Táctil
AmaMedic Hilux 4D tiene tableta remota con pantalla táctil de fácil visualización y soporte magnético. Interfaz amigable para ajustar configuraciones.
